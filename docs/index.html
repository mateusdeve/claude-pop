O que o Claude Pop oferece?

Tudo que você precisa para controlar o Claude Code sem trocar de janela.

🔒

Permissões

Aprove ou negue permissões do Claude Code direto no overlay.

💬

Perguntas

Responda perguntas do Claude sem abrir o terminal.

🔄

Sessões

Troque entre múltiplas sessões do Claude Code instantaneamente.

🔔

Notificações

Receba alertas nativos quando o Claude precisar de atenção.

Como funciona?

Três passos para nunca mais sair do fluxo.

1

Instale o app

Baixe e instale em menos de um minuto. Disponível para macOS, Windows e Linux.

2

Abra o Claude Code

Inicie uma sessão normalmente no terminal. O Claude Pop detecta automaticamente.

3

Use o atalho

Pressione Cmd+J de qualquer lugar para aprovar, responder e gerenciar sessões.

Baixe o
Claude Pop
gratuitamente

Open source, leve e disponível para todas as plataformas.

macOS
Abra o .dmg e arraste para Aplicativos.
Na primeira vez, clique com o botão direito no app → Abrir → confirme no diálogo.
Windows
Execute o Claude Pop Setup.exe. Se o SmartScreen aparecer, clique em Mais informações → Executar assim mesmo.
Linux
Instale com sudo dpkg -i claude-pop_*.deb ou sudo rpm -i claude-pop-*.rpm.
